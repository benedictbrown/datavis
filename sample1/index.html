This page is intended to illustrate some simple javascript usage. It's not a full tutorial, but shows off a few features and quirks, as well as a bit about how javascript interacts with web pages.

  • Press Ctrl-U/Cmd-U to see the HTML source for this page. You'll see that it doesn't include the parts for setting a custom red/green/blue color. That is added dynamically by javascript code.
  • Open the developer tools and select the Elements tab to see the structure of the HTML file after it has been modified by javascript. Look for the <input> elements that were added by javascript code to specify a custom red/green/blue color.
  • Use the Sources pane to see (and debug) the javascript code as well as the HTML and CSS. Pay attention to which parts of this page below the horizontal line are specified in each file.
  • Use the Console pane to see any errors and the output of console.log()statements.
  • Use the Network pane to see network requests, including the information that is sent to the server and the responses received.
